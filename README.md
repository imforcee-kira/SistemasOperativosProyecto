# SistemasOperativosProyecto
Juego Educativo e Interactivo
**************************************************************

Este es el proyecto de Sistemas Operativos realizado por: Ezequiel Rodriguez, Eliel Gonzalez y Ezequiel Forcelledo


**************************************************************
ServerFixer es un juego interactivo, hecho para simular el trabajo de un administrador de sistemas.
El jugador debe diagnosticar, mitigar y recuperar un servidor resolviendo distintos problemas técnicos mediante comandos y decisiones dentro de la terminal.

El proyecto está organizado por niveles, cada uno representando una fase real del mantenimiento de servidores:

Nivel 1 – Diagnóstico: Identificación de fallos y procesos problemáticos.

Nivel 2 – Mitigación: Aplicación de soluciones temporales y control del daño.

Nivel 3 – Recuperación: Restauración del sistema.

Tiene un sistema de estado persistente, un entorno de servidor simulado, scripts de procesos con fallos y una librería de funciones comunes que administran la lógica del juego.

**************************************************************
El juego fue diseñado con la idea de que sea amigable para el usuario y a su vez sea como una practica de los distintos comandos que pueden ser utilizados en el entorno
de Git Bash.
